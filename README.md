# searchable-dropdown-demo
This is a demo of a searchable dropdown component.

### Setup
```
npm install
```

### Compiles for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

# Run unit tests:
```
npm run test:unit
```
